## 📌 Descripción

**Nova Codice** es el sitio web oficial del equipo de ingeniería de la Facultad de Ingeniería en Sistemas, Electrónica e Industrial (**FISEI**). La página presenta a cada integrante del grupo mediante un carrusel interactivo con tarjetas de perfil animadas y efectos visuales modernos.

> _"Si lo puedes imaginar, lo puedes programar."_

---

## 🧩 Características

- 🎨 Diseño futurista con animaciones y efecto *glitch*
- 🌌 Canvas de partículas interactivo en el fondo
- 🃏 Carrusel de tarjetas de miembros del equipo
- 🔗 Perfiles individuales por integrante
- 📱 Diseño responsivo

---

## 🛠️ Tecnologías utilizadas

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura del sitio |
| CSS3 | Estilos y animaciones |
| JavaScript (Vanilla) | Lógica del carrusel y partículas |
| ScrollReveal.js | Animaciones de entrada |
| Google Fonts | Tipografías (Press Start 2P, Rajdhani) |
